## 1. What Meroa is

Meroa is a relationship-first AI companion. Users chat with it in natural language; it
creates and updates tasks and goals from the conversation and tracks progress over time.
Four tabs: **Chat**, **Tasks**, **Goals** (formerly "Tools"), and **You** (settings).

## 2. AI data flow (Apple Guideline 5.1.2(i))

- Chat replies are generated by a **third-party AI service: DeepSeek**
  (`api.deepseek.com`), model **deepseek-v4-flash**. (The provider is configurable; DeepSeek
  is the current production provider.)
- **Data flow:** when a user sends a message, the message **content** — plus server-computed
  context about the user's own tasks and goals (titles, counts) needed to reply — is sent
  over **encrypted HTTPS** to the AI service, which returns a completion. **No phone number
  and no user identifier are sent to the AI service.**
- **Explicit consent is obtained in-app before any message is sent to the AI service**, and
  the server refuses to contact the AI provider until consent is recorded (enforced
  server-side, not just in the UI). Consent is **revocable** in **You → AI data sharing**;
  revoking it stops chat from functioning until re-granted.
- The consent screen appears after sign-in/onboarding and before the user can reach Chat.

## 3. Reporting offensive AI content (Google AI-Generated Content policy)

Users can report an AI response as offensive from within the chat: open the options on an
assistant message → **Report this response** → optional reason. Reports are stored server-
side for review. No additional third-party processing occurs on a report.

## 4. Account deletion & data export

- **In-app deletion:** You tab → **Delete account** (two-step confirm). This performs an
  **immediate hard delete** of all the user's data.
- **Web deletion (Google requirement):** [WEB DELETION URL] — a phone-number + one-time-code
  flow that runs the same hard delete, no app install required.
- **Data export:** You tab → **Export my data** (also available on the web page after code
  verification).
- Deleting the Meroa account does **not** cancel the store subscription; the app tells users
  to cancel in the App Store / Google Play to stop billing.

## 6. Permissions

Meroa requests permissions **only at point of use**:

- **Notifications** — requested when the user enables a reminder, a timer, or a check-in.
  Used for **local** notifications only (no push server, no push token collected).
- No location, contacts, camera, microphone, or photo access is requested (no such features
  exist).
